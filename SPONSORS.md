# Project sponsors

This document records organizations that **sponsor development** of KNXVault
(engineering, maintainership, infrastructure, and open-source stewardship).

Sponsorship here means material support for the project’s development and
community work. It does **not** grant exclusive control over technical
direction or contribution rights; those remain open under the project’s
Apache-2.0 (code) and CC-BY-4.0 (documentation) licenses and contribution
process ([DCO](DCO), [contributing guide](docs/engineering/contributing.md)).

## Adding or updating a sponsor

Project maintainers may update this file when sponsorship relationships change.
Proposed changes should include:

1. Legal entity name  
2. Short description of the sponsorship relationship  
3. Public website URL  
4. Public contact URL (or contact page / security / sales path suitable for open-source inquiries)

For general contribution questions, see [docs/engineering/contributing.md](docs/engineering/contributing.md).
